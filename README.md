# 📉 PocketPath: Actuarial Finance Engine

A Python-powered tool designed to calculate **Burn Rates** and **Spending Risk** using modular logic.

---

### 🚀 Live Logic Status
| Feature | Status | Description |
| :--- | :--- | :--- |
| **Daily Allowance** | ✅ Functional | Calculates spend limits based on time-horizon. |
| **Burn Rate** | ✅ Functional | Predicts survival days based on current velocity. |
| **Risk Alerts** | ✅ Functional | Categorizes spending as 'Within Limit' or 'Overspent'. |
| **Savings Goals** | ⏳ Planned | Logic for target-based capital accumulation. |

### 🛠️ Technical Implementation
- **Language:** Python 3.x
- **Methodology:** Constant-based configuration with `ZeroDivisionError` mitigation.
- **Formatting:** Clean f-string output for financial precision.
